Step 1: Install Python

Make sure you have Python 3.8 or higher installed on your machine.

Step 2: Clone or Download the Project

Download the source code from the repository or copy the files into your project directory.

git clone https://github.com/thienvann12924/Snack-Game.git

Install the Pygame library using pip:

üéÆ User Guide

Game Controls

‚Üë ‚Üì ‚Üê ‚Üí Arrow keys to move the snake
Menu Button Open settings menu
Arrow Keys On/Off Toggle on-screen arrow keys
Speed Control Change game speed (Slow/Normal)
Mute/Unmute Toggle sound

How to Play

  1. Objective: Control the snake to eat food (fruits) to increase score and size
  2. Movement: Use arrow keys or on-screen navigation keys
  3. Avoid Collisions: Do not let the snake run into walls or itself
  4. Score Points: Each time the snake eats food, it grows longer and you gain points
  5. Menu: Click the Menu button to access settings options

Menu Features

  • Arrow Keys Toggle: Show/hide on-screen navigation keys
  • Speed Control: Toggle between slow and normal speed
  • Audio Control: Toggle game sound
